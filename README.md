# Rede Neural Artificial

Desenvolvido como teste de aprendizado na pesquisa(PDPD) sobre Deep Learning e Deep Networks: Modelos, Algoritmos e Aplicações, pela UFABC.
A rede foi pensada para reconhecer digitos, sem bibliotecas de otimização como numpy para multiplicação de vetores propositalmente, porem a integração com o banco de dados do MNIST ainda não foi feita.
Baseado nos conhecimentos obtidos no livro http://neuralnetworksanddeeplearning.com/
